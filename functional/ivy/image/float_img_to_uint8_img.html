float_img_to_uint8_img

ivy.functional.ivy.image.float_img_to_uint8_img(x)[source]

Converts an image of floats into a bit-cast 4-channel image of uint8s, which can be saved to disk.

Parameters

x – Input float image [batch_shape,h,w].

Returns

ret – The new encoded uint8 image [batch_shape,h,w,4] .
