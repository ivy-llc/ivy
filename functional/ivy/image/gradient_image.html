gradient_image¶

ivy.gradient_image(x)[source]¶

Computes image gradients (dy, dx) for each channel.

Parameters

x (array) – Input image [batch_shape, h, w, d] .

Returns

Gradient images dy [batch_shape,h,w,d] and dx [batch_shape,h,w,d] .
