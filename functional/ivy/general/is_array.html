is_array¶

ivy.is_array(x: Any, exclusive: bool = False) → bool[source]¶

Determines whether the input x is an Ivy Array.

Parameters

  • x (any) – The input to check

  • exclusive (bool, optional) – Whether to check if the data type is exclusively an array, rather than a variable or traced array.

Returns

Boolean, whether or not x is an array.
