gather_nd¶

ivy.gather_nd(params: Union[ivy.array.Array, ivy.NativeArray], indices: Union[ivy.array.Array, ivy.NativeArray], dev: Optional[ivy.Device] = None) → Union[ivy.array.Array, ivy.NativeArray][source]¶

Gather slices from params into a array with shape specified by indices.

Parameters

  • params (array) – The array from which to gather values.

  • indices (array) – Index array.

  • dev (ivy.Device, optional) – device on which to create the array ‘cuda:0’, ‘cuda:1’, ‘cpu’ etc. Same as x if None.

Returns

New array of given shape, with the values gathered at the indices.
