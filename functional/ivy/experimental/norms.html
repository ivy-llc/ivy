Norms

ivy.instance_norm(x, /, *, scale=None, bias=None, eps=1e-05, momentum=0.1, data_format='NCHW', running_mean=None, running_stddev=None, affine=True, track_running_stats=False, out=None)[source]

Applies Instance Normalization over a 4D input along C dimension.

Parameters

  • x (Union[Array, NativeArray]) – Input array.

  • scale (Optional[Union[Array, NativeArray]]) – Scale parameter for the normalization. (default: None)

  • bias (Optional[Union[Array, NativeArray]]) – Bias parameter for the normalization. (default: None)

  • eps (float) – Small constant to avoid division by zero. (default: 1e-05)

  • momentum (Optional[float]) – Momentum parameter for running statistics (default: 0.1)

  • data_format (str) – Format of the input data, either ‘NCHW’ or ‘NHWC’. (default: 'NCHW')

  • running_mean (Optional[Union[Array, NativeArray]]) – The running mean of the input array. (default: None)

  • running_stddev (Optional[Union[Array, NativeArray]]) – The running standard deviation of the input array. (default: None)

  • affine (Optional[bool]) – Whether to use affine transformation for the output. (default: True)

  • track_running_stats (Optional[bool]) – Whether to track the running statistics of the input array. (default: False)

  • out (Optional[Array]) – Optional output array, for writing the result to. It must have a shape that the (default: None) inputs broadcast to.

Returns

ret – The normalized array. OR The normalized array, Running mean, Running stddev

Examples

With track_running_stats=False: ret : The normalized array.

>>> x = ivy.eye(3, 3).reshape((1, 3, 3, 1))
>>> ivy.instance_norm(x, scale=[2., 1, 0.5], bias=[2., 1, 0.5], data_format='NCHW',
...                   affine=True, track_running_stats=False)
ivy.array([[[[4.82836342],[0.58581817],[0.58581817]],
        [[0.29290909],[2.41418171],[0.29290909]],
        [[0.14645454],[0.14645454],[1.20709085]]]])

With track_running_stats=True: ret : The normalized array, Running mean, Running stddev.

>>> x = ivy.eye(3, 3).reshape((1, 3, 3, 1))
>>> ivy.instance_norm(x, scale=[2., 1, 0.5], bias=[2., 1, 0.5], data_format='NCHW',
...                   affine=True, track_running_stats=False)
(ivy.array([[[[4.82836342],[0.58581817],[0.58581817]],
        [[0.29290909],[2.41418171],[0.29290909]],
        [[0.14645454],[0.14645454],[1.20709085]]]]),
     ivy.array([[[[0.30000001]],[[0.30000001]],[[0.30000001]]]]),
     ivy.array([[[[0.52426404]],[[0.52426404]],[[0.52426404]]]]))
ivy.l2_normalize(x, /, *, axis=None, out=None)[source]

Normalizes the input array along the given axis to have L2 norm equal to 1.

Parameters

  • x (Union[Array, NativeArray]) – Input array.

  • axis (Optional[int]) – Axis along which to normalize. If None, the whole array is normalized. (default: None)

  • out (Optional[Array]) – optional output array, for writing the result to. It must have a shape that the (default: None) inputs broadcast to.

Return type

Array

Returns

ret – The normalized array.

Examples

>>> x = ivy.array([[1., 2.], [3., 4.]])
>>> ivy.l2_normalize(x, axis=1)
ivy.array([[0.4472, 0.8944],
           [0.6, 0.8]])
