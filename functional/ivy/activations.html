ivy.functional.ivy.activations.leaky_relu(x, alpha=0.2)[source]

Applies the leaky rectified linear unit function element-wise.

Parameters

  • x (Union[Array, NativeArray]) – Input array.

  • alpha (Optional[float]) – Negative slope for ReLU. (default: 0.2)

Return type

Array

Returns

  • ivy.Array – The input array with leaky relu applied element-wise.

  • Examples

  • ———

  • >>> x = ivy.array([0.39, -0.85])

  • >>> y = ivy.leaky_relu(x)

  • >>> print(y)

  • ivy.array([0.39, -0.17])

ivy.functional.ivy.activations.relu(x, out=None)[source]

Applies the rectified linear unit function element-wise.

x

input array

out

optional output array, for writing the result to. It must have a shape that the inputs broadcast to.

Returns
ret

an array containing the rectified linear unit activation of each element in x.

Examples

>>> x = ivy.array([-1., 0., 1.])
>>> y = ivy.relu(x)
>>> print(y)
ivy.array([0., 0., 1.])
Return type

Array

ivy.functional.ivy.activations.softmax(x, axis=- 1)[source]

Applies the softmax function element-wise.

Parameters

  • x (Union[Array, NativeArray]) – Input array.

  • axis (Optional[int]) – The dimension softmax would be performed on. The default is -1 which indicates the last dimension. (default: -1)

Return type

Array

Returns
