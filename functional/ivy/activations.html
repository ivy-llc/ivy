Activations¶

Collection of Ivy activation functions.

ivy.gelu(x, approximate=True)[source]¶

Applies the Gaussian error linear unit (GELU) activation function.

Parameters

  • x (array) – Input array.

  • approximate (bool, optional) – Whether to approximate, default is True.

Returns

The input array with leaky relu applied element-wise.

ivy.leaky_relu(x: Union[ivy.array.Array, ivy.NativeArray], alpha: Optional[float] = 0.2) → ivy.array.Array[source]¶

Applies the leaky rectified linear unit function element-wise.

xivy.Array or ivy.NativeArray

Input array.

alphafloat, default=0.2

Negative slope for ReLU.

ivy.Array

The input array with leaky relu applied element-wise.

Examples: >>> x = ivy.array([0.39, -0.85]) >>> y = ivy.leaky_relu(x) >>> print(y) [0.39, -0.17]

ivy.relu(x: Union[ivy.array.Array, ivy.NativeArray], out: Optional[Union[ivy.array.Array, ivy.NativeArray]] = None) → ivy.array.Array[source]¶

Applies the rectified linear unit function element-wise.

x:

input array

out:

optional output array, for writing the result to. It must have a shape that the inputs broadcast to.

out:

an array containing the rectified linear unit activation of each element in x.

Examples: >>> x = ivy.array([-1, 0, 1]) >>> y = ivy.relu(x) >>> print(y) [-0.0, 0.0, 1.0]

ivy.sigmoid(x: Union[ivy.array.Array, ivy.NativeArray]) → ivy.array.Array[source]¶

Applies the sigmoid function element-wise.

x

input array.

ret

an array containing the sigmoid activation of each element in x.

>>> x = ivy.array([-1., 1., 2.])
>>> y = ivy.sigmoid(x)
>>> print(y)
[0.268, 0.5, 0.880]
ivy.softmax(x: Union[ivy.array.Array, ivy.NativeArray], axis: Optional[int] = - 1) → ivy.array.Array[source]¶

Applies the softmax function element-wise.

x:

Input array.

axis:

The dimension softmax would be performed on. The default is -1 which indicates the last dimension.

out:

The input array with softmax applied element-wise.

Examples: >>> x = ivy.array([-1.0, 0, 1.0]) >>> y = ivy.softmax(x) >>> print(y) [0.09003057, 0.24472847, 0.66524096]

ivy.softplus(x: Union[ivy.array.Array, ivy.NativeArray]) → ivy.array.Array[source]¶

Applies the softplus function element-wise.

x:

input array.

out:

an array containing the softplus activation of each element in x.

Examples: >>> x = ivy.array([-0.3461, -0.6491]) >>> y = ivy.softplus(x) >>> print(y) [0.5350, 0.4203]

Parameters
x:

input array

ivy.Array
