Data type¶

ivy.finfo(type: Union[ivy.Dtype, str, ivy.array.Array, ivy.NativeArray]) → None[source]¶

Machine limits for floating-point data types.

type:

the kind of floating-point data-type about which to get information.

out:

an object having the followng attributes: - bits: int

number of bits occupied by the floating-point data type.

  • eps: float difference between 1.0 and the next smallest representable floating-point number larger than 1.0 according to the IEEE-754 standard.

  • max: float largest representable number.

  • min: float smallest representable number.

  • smallest_normal: float smallest positive floating-point number with full precision.

ivy.iinfo(type: Union[ivy.Dtype, str, ivy.array.Array, ivy.NativeArray]) → None[source]¶

Machine limits for integer data types.

type:

the kind of integer data-type about which to get information.

out:

a class with that encapsules the following attributes: - bits: int

number of bits occupied by the type.

  • max: int largest representable number.

  • min: int smallest representable number.
