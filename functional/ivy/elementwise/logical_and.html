logical_and

ivy.logical_and(x1, x2, *, out=None)[source]

Computes the logical AND for each element x1_i of the input array x1 with the respective element x2_i of the input array x2.

Parameters

  • x1 (Array) – first input array. Should have a boolean data type.

  • x2 (Array) – second input array. Must be compatible with x1. Should have a boolean data type.

  • out (Optional[Union[Array, NativeArray]]) – optional output array, for writing the result to. It must have a shape that the (default: None) inputs broadcast to.

Return type

Array

Returns

ret – an array containing the element-wise results. The returned array must have a data type of bool.
