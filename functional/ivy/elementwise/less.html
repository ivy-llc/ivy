less¶

ivy.less(x1: Union[ivy.array.Array, ivy.NativeArray], x2: Union[ivy.array.Array, ivy.NativeArray]) → ivy.array.Array[source]¶

Computes the truth value of x1_i < x2_i for each element x1_i of the input array x1 with the respective element x2_i of the input array x2.

x1:

first input array. Should have a numeric data type.

x2:

second input array. Must be compatible with x1 (see broadcasting). Should have a numeric data type.

out:

an array containing the element-wise results. The returned array must have a data type of bool.
