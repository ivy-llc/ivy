ivy.default_device(device=None, /, *, item=None, as_native=None)[source]

Returns the input device or the default device. If the as native flag is set, the device will be converted to a native device. If the item is provided, the item’s device is returned. If the device is not provided, the last default device is returned. If a default device has not been set, the first gpu is returned if available, otherwise the cpu is returned.

Parameters

  • device (Optional[Union[Device, NativeDevice]]) – The device to be returned or converted. (default: None)

  • item (Optional[Union[list, tuple, dict, Array, NativeArray]]) – The item to get the device from. (default: None)

  • as_native (Optional[bool]) – Whether to convert the device to a native device. (default: None)

Return type

Union[Device, NativeDevice]

Returns

ret – Device handle or string.

Examples

>>> ivy.default_device()
device(type='cpu')
>>> ivy.default_device("gpu:0")
'gpu:0'
>>> ivy.default_device(item=[], as_native=False)
'cpu'
>>> ivy.default_device(item=(), as_native=True)
device(type='cpu')
>>> ivy.default_device(item={"a": 1}, as_native=True)
device(type='cpu')
>>> x = ivy.array([1., 2., 3.])
>>> x = ivy.to_device(x, 'gpu:0')
>>> ivy.default_device(item=x, as_native=True)
device(type='gpu', id=0)

Set the global split factor for a given device, which can be used to scale batch splitting chunk sizes for the device across the codebase.

Parameters

  • factor (float) – The factor to set the device-specific split factor to.

  • device (Optional[Union[Device, NativeDevice]]) – The device to set the split factor for. Sets the default device by default. (default: None)

>>> ivy.set_backend("torch")
>>> device = torch.device("cuda")
>>> ivy.set_split_factor(0.2)
>>> ivy.set_split_factor(0.3,'gpu')

{'cpu': 0.2, 'gpu': 0.3, device(type='cuda'): 0.4}
Return type

None

ivy.split_func_call(func, inputs, mode, /, *, max_chunk_size=None, chunk_size=None, input_axes=0, output_axes=None, stop_gradients=False, device=None)[source]

Call a function by splitting its inputs along a given axis, and calling the function in chunks, rather than feeding the entire input array at once. This can be useful to reduce memory usage of the device the arrays are on.

Parameters

  • func (Callable) – The function to be called.

  • inputs (Union[Array, NativeArray]) – A list of inputs to pass into the function.

  • mode (str) – The mode by which to unify the return values, must be one of [ concat | mean | sum ]

  • max_chunk_size (Optional[int]) – The maximum size of each of the chunks to be fed into the function. (default: None)

  • chunk_size (Optional[int]) – The size of each of the chunks to be fed into the function. Specifying this arg (default: None) overwrites the global split factor. Default is None.

  • input_axes (Union[int, Iterable[int]]) – The axes along which to split each of the inputs, before passing to the (default: 0) function. Default is 0.

  • output_axes (Optional[Union[int, Iterable[int]]]) – The axes along which to concat each of the returned outputs. Default is same as (default: None) fist input axis.

  • stop_gradients (bool) – Whether to stop the gradients for each computed return. Default is False. (default: False)

  • device (Optional[Union[Device, NativeDevice]]) – The device to set the split factor for. Sets the default device by default. (default: None)

Return type

Union[Array, NativeArray]

Returns

ret – The return from the function, following input splitting and re-concattenation.
