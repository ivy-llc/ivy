eye

ivy.functional.ivy.creation.eye(n_rows, n_cols=None, k=0, dtype=None, device=None)[source]

Returns a two-dimensional array with ones on the k h diagonal and zeros elsewhere.

Parameters

Parameters

  • n_rows (int) – number of rows in the output array.

  • n_cols (Optional[int]) – number of columns in the output array. If None, the default number of columns in the output array is (default: None) equal to n_rows. Default: None.

  • k (Optional[int]) – index of the diagonal. A positive value refers to an upper diagonal, a negative value to a lower diagonal, (default: 0) and 0 to the main diagonal. Default: 0.

  • dtype (Optional[Dtype]) – output array data type. If dtype is None, the output array data type must be the default floating- (default: None) point data type. Default: None.

  • device (Optional[Device]) – device on which to place the created array. (default: None)

  • Returns –

  • ret – device on which to place the created array. Default: None.

Return type

Array
