log2¶

ivy.functional.ivy.array_api.elementwise_functions.log2(x: Union[ivy.array.Array, ivy.NativeArray]) → ivy.array.Array[source]¶

Calculates an implementation-dependent approximation to the base 2 logarithm, having domain [0, +infinity] and codomain [-infinity, +infinity], for each element x_i of the input array x. Special cases For floating-point operands, - If x_i is NaN, the result is NaN. - If x_i is less than 0, the result is NaN. - If x_i is either +0 or -0, the result is -infinity. - If x_i is 1, the result is +0. - If x_i is +infinity, the result is +infinity. Parameters ———- x: array

input array. Should have a floating-point data type.

out: array

an array containing the evaluated base 2 logarithm for each element in x. The returned array must have a floating-point data type determined by type-promotion.
