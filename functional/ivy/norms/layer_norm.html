layer_norm

ivy.layer_norm(x, normalized_idxs, epsilon=1e-05, scale=None, offset=None, new_std=1.0, *, out=None)[source]

Applies Layer Normalization over a mini-batch of inputs

Parameters

  • x (Union[Array, NativeArray]) – Input array

  • normalized_idxs (List[int]) – Indices to apply the normalization to.

  • epsilon (float) – small constant to add to the denominator, use global ivy._MIN_BASE by default. (default: 1e-05)

  • scale (Optional[float]) – Learnable gamma variables for post-multiplication, default is None. (default: None)

  • offset (Optional[float]) – Learnable beta variables for post-addition, default is None. (default: None)

  • new_std (float) – The standard deviation of the new normalized values. Default is 1. (default: 1.0)

  • out (Optional[Array]) – optional output array, for writing the result to. It must have a shape that the (default: None) inputs broadcast to.

Return type

Array

Returns

ret – The layer after applying layer normalization.

Functional Examples

With ivy.Array input:

>>> arr = ivy.full([2, 3], 13, dtype=ivy.float32)
>>> norm = ivy.layer_norm(arr, [0, 1], new_std=2.0)
>>> print(norm)
ivy.array([[0., 0., 0.],
       [0., 0., 0.]])

>>> arr = ivy.array([[1., 2., 3.], [4., 5., 6.]])
>>> norm = ivy.zeros((2, 3))
>>> ivy.layer_norm(arr, [0], out=norm)
>>> print(norm)
ivy.array([[-1., -1., -1.],
       [ 1.,  1.,  1.]])

>>> arr = ivy.array([[0.0976, -0.3452,  1.2740],         [0.1047,  0.5886,  1.2732],         [0.7696, -1.7024, -2.2518]])
>>> norm = ivy.layer_norm(arr, [0, 1], epsilon=0.001,                  new_std=1.5, offset=0.5, scale=0.5)
>>> print(norm)
ivy.array([[ 0.58 ,  0.283,  1.37 ],
       [ 0.585,  0.909,  1.37 ],
       [ 1.03 , -0.628, -0.997]])

With ivy.NativeArray input:

>>> arr = ivy.native_array([[3., 1.],[4., 12.]])
>>> norm = ivy.layer_norm(arr, [0,1], new_std=1.25, offset=0.25, scale=0.3)
>>> print(norm)
ivy.array([[0.2,0.15],[0.225,0.425]])

With a mix of ivy.Array and ivy.Container inputs:

>>> arr = ivy.array([[1., 2., 3.], [4., 5., 6.]])
>>> norm_idxs = ivy.Container({'a': [0], 'b': [1]})
>>> norm = ivy.layer_norm(arr, norm_idxs, new_std=1.25, offset=0.2)
>>> print(norm)
{
    a: ivy.array([[-1.05, -1.05, -1.05],
                  [1.45, 1.45, 1.45]]),
    b: ivy.array([[-1.33, 0.2, 1.73],
                  [-1.33, 0.2, 1.73]])
}

With ivy.Container input:

>>> arr = ivy.Container({'a': ivy.array([7., 10., 12.]),                             'b': ivy.array([[1., 2., 3.], [4., 5., 6.]])})
>>> norm_idxs = ivy.Container({'a': [0], 'b': [1]})
>>> new_std = ivy.Container({'a': 1.25, 'b': 1.5})
>>> offset = ivy.Container({'a': 0.2, 'b': 0.3})
>>> norm = ivy.layer_norm(arr, norm_idxs, new_std, offset)
>>> print(norm)
{
    a: ivy.array([-0.228, 0.0285, 0.199]),
    b: ivy.array([[-0.204, 0., 0.204],
                  [-0.204, 0., 0.204]])
}

Instance Method Examples

Using ivy.Array instance method:

>>> arr = ivy.array([[0.0976, -0.3452,  1.2740],         [0.1047,  0.5886,  1.2732],         [0.7696, -1.7024, -2.2518]])
>>> norm = arr.layer_norm([0, 1], epsilon=0.001,                  new_std=1.5, offset=0.5, scale=0.5)
>>> print(norm)
ivy.array([[ 0.58 ,  0.283,  1.37 ],
       [ 0.585,  0.909,  1.37 ],
       [ 1.03 , -0.628, -0.997]])

Using ivy.Container instance method:

>>> container = ivy.Container(a=ivy.array([0., 1., 2.]), b=ivy.array([3., 4., 5.]))
>>> norm = container.layer_norm([0], new_std=1.25, offset=0.2)
>>> print(norm)
{a: ivy.array([-1.33, 0.2, 1.73]),b:ivy.array([-1.33,0.2,1.73])}

Both the description and the type hints above assumes an array input for simplicity, but this function is nestable, and therefore also accepts ivy.Container instances in place of any of the arguments.
